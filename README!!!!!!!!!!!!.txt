[ゲーム説明] 

時間経過のあるコマンドです。
	会話 - シルヴィと話をします。仲の良さによって話の内容は変わってきます。
	頭を撫でる - シルヴィの頭を撫でます。仲の良さによって反応は変わってきます。
	体を触る - シルヴィの体を触ります。
	ベッドへ - シルヴィをベッドに連れて行き、特殊えっちシーンへ移行します。午前中は選べません。
											特定の服を着せているとベッド以降時コマンドが増えます
	外出 - 一日に一回だけ行えます。
		一人で　-　シルヴィに家の用事を任せて一人で買い出しに行きます。帰りにお土産を買うことができます。
		二人で　-　シルヴィと出かけます。行ける場所はシルヴィとの仲の良さや出かけた時間によって変わります。
			森　-　森の散策をします。なにか役に立つものが見つかるかもしれません。
			街
				喫茶店　-　お昼以降にデザートを食べれます
				お店　-　洋服屋さんです。洋服を買ったり店員さんと話したりできます
						（シルヴィとの親愛度が高いとまとめ買いができるようになります
				市場　-　市場を見て回ります。
	お茶を淹れる - シルヴィとお茶をします。お茶請けがあるとそれも一緒に食べます。
	仕事を手伝ってもらう　-　ある衣装を着せているとシルヴィにお仕事を手伝ってもらえます。

時間経過のないコマンドです。
	調合 - 材料があれば薬を作れます
	着替え - シルヴィに着替えてもらいます。基本的に服は先にお店で買っておかないといけません。
	呼び名変更 - シルヴィに呼び方を変えてもらいます。
	シルヴィステータス - シルヴィの状態やどう思われているかを知ることができます。
	生活記録 - シルヴィと過ごした日数や行動回数などの記録が見れます。
	設定 - 各種設定を行います
	回想 - 一度見たえっちシーンを見直すことができます。また解放されてないシーンは解放条件のヒントが確認できます。


画面のサイズが合わない方はpackage.jsonをメモ帳やエディエタで開きwidthとheightの数値を変更することで画面サイズを変えることができます。
